How to Run Unit Tests
Prerequisites
Python 3.6 or higher
pytest

You can install pytest using pip:
pip install pytest

Running the Unit Tests
Navigate to the 2_unit_testing directory.

Run the pytest command:
pytest

The tests will run, and you will see the results in your terminal.

Test Structure
test_auth.py: Contains unit tests for authentication.
test_inventory.py: Contains unit tests for inventory management.
test_orders.py: Contains unit tests for order management.
